# HM BRAKES e-Commerce (Herencia en POO)

Proyecto académico que demuestra el uso de herencia en JavaScript ES6 para una plataforma e-Commerce.

## Ejecutar

1. Instalar Node.js
2. Abrir la carpeta en Visual Studio Code
3. Ejecutar:

npm install
npm start

## Funcionalidades

- Herencia entre clases
- Productos físicos y digitales
- Usuarios Cliente y Administrador
- Gestión básica de inventario
